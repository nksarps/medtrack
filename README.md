# Data Quality Assessment Report - MedTrack Ghana

## Overview
This repository contains a comprehensive **Data Quality Assessment Report** conducted for MedTrack Ghana, a health technology startup. The report analyzes a sample patient appointment dataset to identify data quality issues that affect operations, clinical care, and billing processes.

The assessment focuses on six key data quality dimensions:

- Accuracy  
- Completeness  
- Consistency  
- Timeliness  
- Validity  
- Uniqueness  

The report also evaluates the business impact of each issue and provides actionable recommendations to resolve critical problems.
